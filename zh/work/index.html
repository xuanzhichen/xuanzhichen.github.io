2023. 09 – 2024. 04

A Primer on Causal Diagram Learning (科普工作)

  • 因为我是从一本名为《为什么》(The Book of WHY) 的科普书中获得因果关系的基本思想启发的，因而围绕着“因果图学习”的主题，我试图在这本书中进一步建立它与因果领域先驱们所著作的其他专业书籍的联系。

  • 我希望告诉我的读者们：当站上这些巨人的肩膀上时，我们将看到因果发现的思想是深深植根于过去的。当然,本文引入的因果图学习的思想仅是冰山一角，但我希望引领者们在因果科学领域的研究观点或许能对人工智能，认知神经科学，以及其他相关领域的未来发展有所裨益。

  • 相关论文(本页): “A Primer on Causal Diagram Learning: Interpreting Causation from the Causal Discovery Perspective, 2024.”

2022.05 – 2023.06

Cadimulc: Light Python Package for Hybrid-Based Causal Discovery (软件开发)

  • CADIMULC 意为 CAusal DIscovery with Multiple Latent Confounders（多隐变量下的因果关系发现）, 是一种提供基于混合模式因果发现的 Python 软件包。

  • 作为一款轻量级软件包，它提供了开箱即用的 Python 接口，以相对高效的方式从一般的原始数据中学习经验性因果结构。该软件包集成了混合型因果发现方法 (hybrid-based causal discovery approaches) 的实现（包括 MLC-LiNGAM 算法），以及因果发现任务的 “微型” 工作流——如数据生成，结果评估，和因果图可视化。

2021.11 – 2023.06

Nonlinear Causal Discovery from Unknown Common Causes (科研课题)

  • 我们能否教会脑科学领域的人工智能通过数据中蕴含的特殊可识别性 (specific identification) 来操纵因果关系?在复杂的学习环境中, 我们应该如何进一步理解数据背后潜藏的“因果结构 (causal structures)”，尤其是当智能体置身于复杂学习环境中——在这样的环境中通用的数据关系往往倾向于是非线性 (non-linear) 的,甚至来自多个未知因素 (multiple unknown factors) 的影响都是持续不断的。

  • 针对该问题的现有解决方案要么在形式化的理论表示中难以捉摸，要么在算法的计算层面上难以实现。这样的动机驱使我们找到一种基于理论指导的，并且在实际中表现高效的因果发现算法。

  • 相关论文(本页): “Non-linear Causal Discovery for Additive Noise Model with Multiple Latent Confounders, 2023.”
