项目介绍

2021年9月 – 至今

Nonlinear Causal Discovery from Potential Confounding (科研课题)

从含有潜在混淆的原始数据中发现因果关系

  • 因果发现 (Causal discovery)，即以因果图 (causal graphs) 的形式粗略地恢复 (recover) 数据的生成机制，是一门从观测性静态数据中发现因果蕴意的新兴的学科。

  • 在实践中，往往只有一个相对于合理数据集的变量的子集被观测到，导致了由这些隐变量 (latent variables) 的引起的潜在混淆 (potential confounding); 与此同时，由于变量之间一般的非线性相互作用，在线性情况下行之有效的因果约束 (causality constraints) 准则将面临阻碍。

  • 详见本页面 Papers.2023 的 “Non-linear Causal Discovery for Additive Noise Model with Multiple Latent Confounders”.

2023年7月 – 2024年2月

A Brief Introductory for Causal Diagram Learning (科普工作)

关于如何学习因果图的简介

  • 在该简介中，我尝试将 反事实 (counterfactuals) 和干预 (intervention) 的概念 (人类因果思维的范式) 引入关于重大问题因果关系分析的两个 (简化) 应用中：例如全球气候变化和新冠肺炎。

  • 介绍给定一个因果图的 “下游” 作用，并回溯性地介绍 “上游” 任务，即 “因果发现” : 关注如何学习一个特定的因果图。

  • 详见本页面 Papers.2023 的 “A Primer on Causal Diagram Learning: Interpreting Causation from the Causal Discovery Perspective”.

2022年5月 – 2024年2月

Cadimulc: Light Python Package for Hybrid-Based Causal Discovery (软件开发)

Cadimulc: 一种提供基于混合模式因果发现的 Python 软件包

  • CADIMULC 意为 CAusal DIscovery with Multiple Latent Confounders（多隐变量下的因果关系发现），作为轻量级软件包，cadimulc 提供了开箱即用的 Python 接口，以相对高效的方式从一般的原始数据中学习经验性因果结构。

  • 该软件包集成了基于混合型方法的实现（包括 MLC-LiNGAM 算法），以及因果发现任务中的 “微型” 工作流——例如数据生成，结果评估，以及因果图可视化。

  • Chen, XZ., Chen, W., Cai, RC. Non-linear Causal Discovery for Additive Noise Model with Multiple Latent Confounders. Xuanzhi’s Personal Website. [pdf] [video] [code]

    摘录：在复杂学习的环境中 (如非线性关系，众多未知因素)，我们应该如何进一步抵达对于大脑数据背后 (因果) 结构的理解? 现有的解决方法或者在理论表达上显得捉摸不透，或者在实践在计算上已是众所周知的困难……我们通过重新分析关于非线性加性噪声因果模型 (Additive Noise Models, ANMs) 发现了一个基本准则，并籍次开发了一种新型的基于混合模式的因果发现算法。

2022

  • Liu, YQ.*, Zhu, WH.*, Qiao, J.*, Huang, ZY., Xiang, Yu., Chen, XZ., Chen, W., Cai, RC. Causal Alignment Based Fault Root Causes Localization for Wireless Network. ICASSP; IEEE International Conference on Acoustics, Speech and Signal Processing. [pdf]

    摘录：尽管有监督方法 (supervised methods) 在训练样本中显示出饶有希望的结果，但大多数现有方法都依赖于假设训练和测试样本独立同分布。该假设通常被由跨领域的设备不一致性而引起的网络故障所打破，即分布偏移 (distribution shift) 问题。因此，有必要对齐训练集和测试集的分布。
